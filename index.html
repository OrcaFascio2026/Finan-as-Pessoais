💰 Finanças Pessoais

Configure uma vez, acesse de qualquer lugar.

  1. 1
    Crie uma conta grátis em supabase.com e crie um novo projeto.
  2. 2
    No painel do Supabase, vá em SQL Editor e execute: CREATE TABLE transactions ( id uuid DEFAULT gen_random_uuid() PRIMARY KEY, user_id text NOT NULL, type text NOT NULL CHECK (type IN ('income','expense')), description text NOT NULL, amount numeric NOT NULL, category text, date date NOT NULL, created_at timestamptz DEFAULT now() ); Em seguida, habilite Row Level Security (RLS) em: Table Editor → transactions → RLS → Enable.

    Adicione esta política RLS: CREATE POLICY "user_own_data" ON transactions USING (user_id = current_setting('app.user_id', true)) WITH CHECK (user_id = current_setting('app.user_id', true));
  3. 3
    Copie a URL e a anon key do projeto (Settings → API) e cole abaixo.
